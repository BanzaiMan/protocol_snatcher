Mail.app URL Rewriter

Mail.app URL Rewriter (MURLR) lets you edit any URL you click in your Mail.app message view. This can be handy if you correspondents send you links to documents on the local area network.

Rules

Rewrite rules are ordered. The first rule that matches the clicked link will trigger an event. If the URL doesn't match any rule, the click is passed on verbatim to Mail.app, just as though you did not have MURLR.

Note that the replacement occurs globally.

Properties

Match Regular Expression
An oniguruma library regular expression to match against the clicked link.
Replacement Text
The text that replaces the match. Usual regular expression tricks like back references apply. While it is permissible to have an empty string as replacement text, it is utterly useless to do so.
Remote Volume (Optional)
If this property is empty, MURLR will pass the re-written URL to Mail.app.
If this property is not empty, MURLR will mount the volume and open the file specified by the re-written path (so you probably want to specify the replacement text that starts with /Volumes).
