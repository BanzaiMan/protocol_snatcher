Mail.app URL Rewriter

Mail.app URL Rewriter (MURLR) lets you any URL you click in your Mail.app message view. This can be handy if you correspondents send you link to a shared local area network.

Rules

Rewrite rules are an ordered list. The first rule that matches the clicked link will trigger an event. If the URL doesn't match any rule, the click is passed on verbatim to Mail.app, just as though you did not have MURLR.

Properties

Match Regular Expression
A oniguruma library regular expression to match against the clicked link.
Replace Text
The text that replaces the URL. Note that you can use numbered captures in replacement text.
Remote Volume (Optional)
If this property is empty, MURLR will pass the re-written URL to Mail.app. If this property is not empty, MURLR will attempt to mount it, and then opens the file specified by the re-written URL.
