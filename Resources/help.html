Rules

Rewrite rules are an ordered list. The first rule that matches the clicked link will trigger an event. If no match happens, the click is passed on verbatim to the OS, just as though you did not have MURLR.

Properties

Match Regular Expression
A Perl regular expression to match against the clicked link
Replace Text
The text that replaces the URL. Just as in Perl, you can use numbered captures in replacement text.
Remote Volume (Optional)
The remote server volume to mount.
